# Luvin CMS

## How to use

### Dev

Start app

```bash
yarn start
# or
npm start
```

### Deploy app

1: Install serve package

```bash
yarn global add serve
#or
npm install -g serve
```

2: Build

```bash
yarn build
# or
npm run build
```

3: Start

```bash
serve -s build -l 4000
```

4000 = PORT
